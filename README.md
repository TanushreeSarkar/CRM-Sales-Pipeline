📊 CRM Sales Pipeline – Java Spring Boot (Maven)
A powerful and scalable Customer Relationship Management (CRM) system built with Java, using Spring Boot and Maven. Designed to manage the complete sales cycle — from capturing leads to closing deals — with integrated dashboards and automation to improve sales productivity.

🛠 Tech Stack
Backend: Java 17, Spring Boot, Spring Data JPA, Spring Security

Build Tool: Maven

Database: MySQL / PostgreSQL (configurable)

Authentication: JWT-based login

Cloud Integration: Firebase / AWS SDK

Frontend: (Optional - React / Thymeleaf if used)

Deployment: Docker / Heroku / AWS EC2

🚀 Features
Lead & Opportunity Management: Track and manage leads throughout the sales lifecycle.

Sales Pipeline Visualization: Intuitive UI with sales stages and deal progress.

User Authentication & Roles: Secure access using JWT; admin, manager, and sales roles.

Real-Time Notifications: Automated follow-ups and activity reminders.

Custom Dashboards: Sales performance insights, KPIs, and conversion rates.

RESTful APIs: Modular, documented API endpoints (Swagger/OpenAPI ready).

Cloud-Enabled: Firebase or AWS-based storage and analytics integration.

Logging & Monitoring: Integrated with SLF4J/Logback and actuator endpoints.

🎯 Business Impact
Boosted sales team efficiency with a 30% increase in productivity.

Enabled real-time collaboration and better conversion tracking.

Scalable solution suitable for startups and growing businesses.
